# Edificación y Asesoría Integral - Landing page

Página web estática basada en la referencia enviada por WhatsApp/Facebook.

## Archivos

- `index.html`: estructura de la página.
- `styles.css`: diseño responsive.
- `script.js`: menú móvil y animaciones al hacer scroll.
- `assets/logo.png`: logo recortado de la referencia.
- `assets/hero.jpg`: imagen principal recortada de la referencia.
- `assets/preview.jpg`: imagen para compartir en redes.

## Cómo usar

Abre `index.html` en el navegador o sube todos los archivos a tu hosting conservando la misma estructura de carpetas.
